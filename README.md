## 자주 사용하는 수학 공식 정리

✅ 1. 1부터 n까지의 합
n(n + 1) / 2
✅ 2. 1부터 n까지의 제곱 합
n(n + 1)(2n + 1) / 6
✅ 3. 1부터 n까지의 세제곱 합
(n(n + 1) / 2)²
✅ 4. 홀수의 합 (n 이하)
((n + 1) / 2)²
✅ 5. 짝수의 제곱 합 (n 이하 짝수만)
n(n + 1)(n + 2) / 6
✅ 6. 등차수열 합 (a부터 시작, d만큼 증가, n개 항)
n/2 \* (2a + (n - 1)d)
✅ 7. 두 수 a, b의 최소공배수 (LCM)과 최대공약수 (GCD)
GCD(a, b): 유클리드 호제법 → 재귀로 gcd(a, b) = gcd(b, a % b)
공식: b === 0 ? a : gcd(b, a % b)
LCM(a, b): (a \* b) / gcd(a, b)
